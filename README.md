# Design System Roadmap

A responsive, interactive roadmap for building a design system. This project allows teams and individuals to track progress across phases of creating a design system, from basics to advanced topics.

## Features

- Interactive checklist with subtasks.
- Dynamic progress tracking with progress bar.
- Save roadmap as PDF or image.
- Reset progress functionality.
- Light and dark mode toggle.
- Fully responsive layout for desktop, tablet, and mobile.
